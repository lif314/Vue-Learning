## 总结

- 组件化编码流程：
    - 拆分静态组件：组件要按照功能点拆分，命名不要与html元素冲突。
    - 实现动态组件：考虑好数据的存放位置，数据是一个组件在用，还是一
些组件在用：
        - 一个组件在用：放在组件自身即可。
        - 一些组件在用：放在他们共同的父组件上（状态提升）。
    - 实现交互：从绑定事件开始。
- props适用于：
    - 父组件=>子组件通信
    - 子组件=>父组件通信（要求父先给子一个函数）

- 使用v-mode时要切记：v-mode绑定的值不能是props传过来的值，因props是不可以修改的！

- props传过来的若是对象类型的值，修改对象中的属性时Vue不会报错，但不推荐这样做。
